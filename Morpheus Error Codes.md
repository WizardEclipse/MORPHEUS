# Morpheus Error Codes:

## Index:
- 0xx - Comms
- 1xx - GPIO
- 2xx - PWM
- 3xx - ADC
- 4xx - Fatal Error

#### 0xx Error Codes:
- 001 - Paulo to Testi handshake failed
- 002 - Testi to Paulo handshake failed
- 003 - UART time out

#### 1xx GPIO Error Codes:
- 1(00 - 28) - GPIO pin test failure

#### 4xx Fatal Error Codes:

- 400 -  Flashing Failure
